JSON to Query Params

Para esta kata vamos a TRANSFORMAR un JSON a una URL con QUERY PARAMS.
Los query params son parametros que se le adicionan a una ruta de un recurso web determinado.
Por ejemplo para buscar sobre el recurso localhost:8080/persona por nombre, apellido
le debemos adicionar dos query params para permitir la busqueda
y quedaria algo asi localhost:8080/persona?nombre=Matias&apellido=Icardi
